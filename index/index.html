How do I launch it?

You need to have Python and install the 'requests' library.

• Windows: pip install requests

• Ubuntu / Debian / Mint: sudo apt update && sudo apt install python3-requests

• Fedora: sudo dnf install python3-requests

• Arch Linux / Manjaro: pacman -S python-requests

• MacOS: pip3 install requests
